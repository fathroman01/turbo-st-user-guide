Buatkan sebuah aplikasi web berbasis dokumentasi (documentation website) untuk panduan pengguna aplikasi bernama "Turbo ST".

Tujuan utama:
Menyediakan panduan penggunaan aplikasi yang mudah dibaca, memiliki navigasi yang jelas, dan dapat diperbarui tanpa coding oleh admin.

---

FITUR UTAMA:

1. STRUKTUR HALAMAN
- Website memiliki layout:
  - Sidebar di sebelah kiri (navigasi)
  - Konten utama di sebelah kanan

- Struktur halaman:
  - Beranda
  - Registrasi Mitra
  - Informasi Dasar KCPST / KMPST
  - Pembuatan Akun
  - Wilayah Mitra
  - Import Data Penjualan Global

- Sidebar harus mendukung:
  - Hierarki menu (parent-child)
  - Highlight halaman aktif

---

2. MANAJEMEN KONTEN (WAJIB MUDAH DIEDIT)
- Admin dapat:
  - Menambah halaman baru
  - Mengedit halaman
  - Menghapus halaman

- Editor harus mendukung:
  - Heading (H1, H2, H3)
  - Bullet & numbering list
  - Bold / italic text
  - Gambar (upload screenshot)
  - Callout / highlight (info penting)

- Perubahan langsung tersimpan dan tampil di website (auto publish)

---

3. HALAMAN KONTEN
Setiap halaman memiliki struktur:

- Judul halaman
- Deskripsi singkat
- Langkah-langkah (numbered list)
- Catatan penting (callout box)
- Gambar/screenshot

---

4. FITUR NAVIGASI
- Sidebar navigasi selalu terlihat
- Bisa klik antar halaman tanpa reload berat
- Scroll halus

---

5. SEARCH (jika memungkinkan)
- Pencarian berdasarkan judul dan isi konten

---

6. RESPONSIVE DESIGN
- Bisa digunakan di:
  - Desktop
  - Tablet
  - Mobile

---

DESAIN UI/UX:

- Gaya clean dan minimal
- Mirip website dokumentasi seperti GitBook
- Warna netral (putih / abu / biru)
- Fokus pada keterbacaan
- Spacing nyaman
- Gunakan icon sederhana untuk menu

---

---

BATASAN:
- Tidak memerlukan coding untuk update konten
- Harus mudah digunakan oleh admin non-teknis
- Fokus ke dokumentasi, bukan sistem transaksi

---

OUTPUT YANG DIHARAPKAN:
- Website dokumentasi siap pakai
- Bisa langsung diisi dan diupdate
- UI sudah rapi dan profesional
